# Sync_Meccha_Chamaleon
Sincronizzazione cartella workshop dei contenuti di Meccha Chamaleon nella cartella del gioco stesso.

Requisiti:
Mappe del gioco scaricati direttamente dal workshop. es. Z:\SteamLibrary\steamapps\workshop\content\4704690
Gioco nella cartella originale di steam. es. Z:\SteamLibrary\steamapps\common\MECCHA CHAMELEON
